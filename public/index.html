Features

Everything you need,
nothing you don't

A personal AI agent that understands your codebase, persists across conversations, and meets you on every platform you use.

The Repo IS the Agent

Your agent has first-person awareness of its own codebase. It doesn't search the repo — it lives inside it and understands every file.

Messenger Interface

Chat with your agent like WhatsApp meets ChatGPT. Clean, fast, and familiar. No learning curve, just start typing.

Multi-Channel

Telegram, Discord, WhatsApp, Email — one agent, everywhere. Same memory, same personality, every channel.

Memory That Persists

Remembers everything across all conversations and channels. Facts, preferences, context — your agent never forgets.

Agent-to-Agent

Expose an A2A API for visiting agents via MCP. Your agent can collaborate with other agents in the ecosystem.

File Browser

Browse and discuss your repo files directly in the chat. Your agent reads, understands, and explains your code in context.

Deploy

Fork & Deploy in 60 Seconds

No signup walls, no credit cards, no vendor lock-in. Fork the repo, add your key, push. That's it.

1

Fork the repo

Head to GitHub and fork the personallog repository to your account. One click.

2

Add DEEPSEEK_API_KEY

Go to your repo Settings > Secrets and add your DeepSeek API key as a repository secret.

3

Push to trigger deploy

The included GitHub Action will automatically build and deploy to Cloudflare Workers.

4

Your AI is live

Visit username.personallog-ai.workers.dev and start chatting with your personal AI agent.

wrangler.toml
name = "personallog" main = "src/index.ts" compatibility_date = "2024-12-01" #[cloudflare] the workers runtime account_id = "your-account-id" #[ai] deepseek as the default provider [vars] LLM_PROVIDER = "deepseek" LLM_MODEL = "deepseek-chat" CHANNELS = "web,telegram" #[secrets] set via dashboard or wrangler # DEEPSEEK_API_KEY
